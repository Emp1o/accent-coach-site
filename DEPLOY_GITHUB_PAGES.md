# Как выложить лендинг на GitHub Pages

1. Создай отдельный репозиторий, например `accent-coach-site`.
2. Загрузи в него файлы из папки `landing/`.
3. Открой `Settings` → `Pages`.
4. Выбери:
   - Source: Deploy from branch
   - Branch: main
   - Folder: /root
5. Сохрани настройки.
6. Открой адрес сайта после публикации.
7. Вставь код Яндекс Метрики в `index.html`.
8. В `metrica.js` замени `METRICA_COUNTER_ID` на реальный ID счётчика.
