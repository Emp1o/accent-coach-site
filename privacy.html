Privacy Policy

Политика конфиденциальности сайта и локальной версии расширения ОрфоДракон.

1. Какие данные может собирать сайт

Лендинг может использовать Яндекс Метрику для сбора обезличенной статистики посещений, источников трафика и кликов по кнопкам установки и перехода на GitHub.

2. Какие данные хранит расширение

Локальная версия расширения может хранить в браузере пользователя карточки, настройки, историю повторений и результаты тренировок. Эти данные используются только для работы расширения и обучения пользователя.

3. Передача данных третьим лицам

В текущей локальной версии проекта пользовательские тексты не обязаны отправляться на внешний сервер для базовой работы расширения. Если в будущих версиях появится серверная обработка, эта политика будет обновлена отдельно.

4. Персональные данные

Сайт и локальная версия расширения не предназначены для сбора паспортных данных, банковских данных, паролей или иных чувствительных персональных данных.

5. Контакты

По вопросам проекта можно использовать страницу репозитория: github.com/Emp1o/accent-coach-extension.
