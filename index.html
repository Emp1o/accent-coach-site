Установка

Как установить расширение

Сейчас основной путь — через GitHub Release. Позже можно добавить публикацию в Chrome Web Store.

Основной способ — через GitHub

  1. Нажми Скачать расширение.
  2. Распакуй архив на компьютер.
  3. Зайди в более подходящий браузер (Google или Yandex), в поисковой строке в Google набери chrome://extensions, в Yandex - browser://extensions.
  4. Включи Режим разработчика (Developer mode).
  5. Нажми Загрузить распакованное расширение (Load unpacked).
  6. Выбери папку extension в распакованном архиве.

Когда появится версия в магазине

После публикации в Chrome Web Store установка станет проще: достаточно будет нажать одну кнопку в магазине и добавить расширение в браузер.

FAQ

Частые вопросы

Расширение бесплатное?

Да, базовая локальная версия может работать без внешнего API.

Это безопасно?

Да. Код проекта открыт на GitHub, а карточки и настройки в локальной версии хранятся в браузере пользователя.

Почему установка идёт через Developer mode?

Потому что текущая публичная версия распространяется через GitHub Release. После публикации в Chrome Web Store путь установки станет проще.

Подходит для подготовки к ЕГЭ?

Да. Особенно полезны карточки, тренировки, уведомления и отдельный ЕГЭ-режим.

Можно ли посмотреть исходный код?

Да, для этого на сайте есть отдельная кнопка GitHub.
