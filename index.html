Как это работает

Сценарий использования

  1. Открываешь расширение или выделяешь текст на странице.
  2. Проверяешь ударение или пунктуацию.
  3. Сохраняешь сложные случаи в карточки.
  4. Повторяешь через тренировки и уведомления.
  5. Перед экзаменом запускаешь ЕГЭ-режим.
Почему это удобно

Не просто справочник, а тренажёр

  • Не надо прыгать между словарём, заметками и тестами.
  • Сложные слова и предложения собираются в одном месте.
  • Карточки и ЕГЭ-режим помогают не просто посмотреть, а реально запомнить.
Установка

Как установить

Выбери удобный вариант. Для обычных пользователей лучше Chrome Web Store. Для open-source установки — GitHub.

Вариант 1 — через Chrome Web Store

  1. Нажми кнопку Установить расширение.
  2. Открой страницу расширения в магазине.
  3. Нажми Add to Chrome.
  4. Закрепи расширение на панели браузера.

Вариант 2 — через GitHub

  1. Открой GitHub по кнопке выше.
  2. Скачай ZIP или открой Releases.
  3. Распакуй архив.
  4. Открой chrome://extensions.
  5. Включи Developer mode.
  6. Нажми Load unpacked.
  7. Выбери папку extension.

FAQ

Частые вопросы

Расширение бесплатное?

Да, базовую локальную версию можно использовать бесплатно.

Подходит для подготовки к ЕГЭ?

Да. Особенно полезны карточки, режим тренировки и ЕГЭ-режим.

Нужно ли что-то устанавливать дополнительно?

Если используешь версию из магазина — обычно нет. Если ставишь через GitHub, нужна ручная установка как unpacked extension.

Можно ли посмотреть исходный код?

Да, для этого на лендинге есть отдельная кнопка GitHub.

Это безопасно?

Да, код расширения открыт и доступен на GitHub. Ты можешь посмотреть, что именно устанавливаешь.

Почему установка идёт через Developer mode?

Потому что текущая версия распространяется через GitHub. После публикации в Chrome Web Store установка станет проще.

Работает ли в Яндекс Браузере?

Да, если браузер поддерживает Chromium-расширения и ручную установку через страницу расширений.
